## Run tests

```sh
mvnw verify
```

# To run locally
1) Run
   ```sh
   mvn package
   ```
2) Run the Spring Boot application:
   ```sh
   mvn spring-boot:run
   ```
4) Take a look at http://localhost:8080/swagger-ui.html.
